# Android团队小工具客户端
## 设备信息登记
* 标识设备所属 （公司/个人）
* 标识设备当前使用人
## 上传设备信息
* Device ID
* Android ID
* Serial Number
* Mac Address
* Phone Type
* SDK Version
* OS Version
* Density
* Screen Width
* Screen Height
* 所属信息
  * 公司/个人
  * 使用者姓名
## 产品新测试版本提示
* 每次团队里某个产品发布新的测试版本，收到推送提醒
* 点击推送提醒，下载apk
## 产品列表
* 查看团队所有产品的列表
## 二维码扫描
* 链接：打开相应地址
* 非链接： 显示文字
